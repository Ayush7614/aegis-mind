# LLM Red Teaming

Automated and manual red teaming for LLM applications.

## Stack

| Tool | Use case |
|------|----------|
| [Promptfoo](https://github.com/promptfoo/promptfoo) | Eval suites, regression, CI gates |
| [Deepteam](https://github.com/confident-ai/deepteam) | Adversarial test generation |
| Ollama / local models | Safe lab targets |
| Custom probes | Domain-specific injection sets |

## Workflow

```text
Define policies → Write probes → Run eval → Triage failures → Patch → Re-run
```
