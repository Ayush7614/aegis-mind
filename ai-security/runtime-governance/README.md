# Runtime Governance

Securing autonomous agents at execution time — sandboxes, replay, policy enforcement.

## Patterns

- **Sandboxed execution** — isolated containers per agent session
- **Tool allowlists** — explicit MCP capability grants
- **Session replay** — audit what the agent did and why
- **Egress control** — block arbitrary fetch / SSRF from tools
- **Human approval gates** — sensitive actions require step-up
