# Templates

## Note template

See [docs/getting-started.md](../../docs/getting-started.md#note-template).

## Bug bounty report skeleton

```markdown
# Title: [Product] — [Vuln class] — [Impact]

## Summary
2-3 sentences. What breaks, who cares.

## Steps to reproduce
1. ...
2. ...

## Impact
Concrete harm — data accessed, account taken over, etc.

## Supporting material
Screenshots / video (redacted).

## Remediation
Specific fix recommendation.
```
