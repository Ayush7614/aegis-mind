# Bug Bounty

Hunting workflow, validation gates, and report writing for H1 / Bugcrowd / Intigriti.

## Workflow

```text
Recon → Surface map → Hunt class → Validate → Report → Triage
```

## Core principles

1. **Prove impact** — no "could potentially" submissions
2. **7-question validation gate** before writing
3. **Evidence hygiene** — redact cookies and PII
4. **Chain thinking** — IDOR + missing step-up = higher impact
